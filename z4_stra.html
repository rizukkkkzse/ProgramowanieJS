Teoria JavaScript - Strafun

1) Kiedy powstał JavaScript (podaj datę).

JavaScript powstał w grudniu 1995 roku.

2) Dlaczego jest to język skryptowy?

Jest to język skryptowy, ponieważ działa w środowisku interpretera i nie wymaga kompilacji.

3) Gdzie jest wykonywany JS (klient lub serwer)?

JavaScript jest wykonywany głównie na kliencie (w przeglądarce), choć może działać również na serwerze (np. Node.js).

4) Dlaczego JS to bezpieczne narzędzie?

JavaScript działa w tzw. sandboxie, co ogranicza jego dostęp к systemowym zasobам użytkownika.

5) Czy jest to język obiektowy? Jeśli tak, to dlaczego.

Tak, JavaScript jest językiem obiektowym, ponieważ obsługuje tworzenie i manipulację obiektами.

6) Wymień zalety JS:

  • Zaleta 1: Działa w przeglądarce bez dodatkowego oprogramowania.
  • Zaleta 2: Dynamiczny i elastyczny.
  • Zaleta 3: Bogaty ekosystem bibliotek i frameworków.

7) Framework to:

Zbiór narzędzi i bibliotek ułatwiających rozwój aplikacji.

8) Do aplikacji mobilnych można użyć Framework.

Tak, np. React Native.

9) Aplikacje responsywne to:

Aplikacje, które dostosowują się do różnych rozmiarów ekranów.

10) ‹input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')"›

Tworzy przycisk, który при нажатии otworzy новое окно с указанным содержимым.

11) window.open()

Otwiera nowe okno przeglądarki z określonymi opcjami, np. wysokość, szerokość, brak paska narzędzi.

12) window.close()

Zamyka aktywne okno przeglądarki.
